🔌 API ENDPOINTS
================================================================================

✅ POST /api/auth/register
   - Create new user account
   - Input: username, email, password, user_type, display_name, city
   
✅ POST /api/auth/login
   - User login with email, phone, or username
   - Input: email/phone/username, password
   
✅ GET /api/auth/verify-email?token=xxx
   - Verify email with token
   - 24-hour token expiry

✅ GET /api/admin/stats
   - Get admin dashboard statistics
   - Protected: Requires admin JWT token
   - Response: users, escorts, clients, advertisers, bookings, pendingVerifications
