## COLOR SCHEME

```
🎨 PRIMARY COLORS
  • Primary (Red):      #EF4444 - Buttons, CTAs, highlights
  • Secondary (Pink):   #EC4899 - Accents, hover states
  • Dark BG:           #111827 - Main background
  • Card BG:           #1F2937 - Card/section background
  • Text Light:        #E5E7EB - Primary text
  • Text Muted:        #9CA3AF - Secondary text

🏷️ TIER BADGES
  • Elite:      #DC2626 (Red)
  • VIP:        #EA580C (Orange)
  • Premium:    #EAB308 (Yellow)
  • Standard:   #6B7280 (Gray)

✅ STATUS COLORS
  • Success:    #10B981 (Green)
  • Error:      #EF4444 (Red)
  • Warning:    #F59E0B (Orange)
  • Info:       #3B82F6 (Blue)
```

## TYPOGRAPHY

```
📝 HEADINGS
  • h1: 28-32px, Bold (2xl)
  • h2: 20-24px, Bold (xl)
  • h3: 16-18px, Bold (lg)
  • h4: 14-16px, SemiBold (base)

📄 BODY TEXT
  • Desktop: 14px, line-height 1.5
  • Mobile: 14px, line-height 1.6
  • Small: 12px
  • Tiny: 11px (labels)

🔤 FONT STACK
  • Sans-serif: System fonts, Geist fallback
  • Mono: Consolas, Monaco (code/technical)
```

## RESPONSIVE BREAKPOINTS

```
📱 MOBILE
   Width: 320px - 640px
   Sidebar: Icon only (w-20)
   Carousel: 2-3 cards
   Grid: 2 columns
   Font: 14px body

📱 TABLET
   Width: 640px - 1024px
   Sidebar: Toggle available
   Carousel: 3 cards
   Grid: 3 columns
   Font: 14px body

💻 DESKTOP
   Width: 1024px+
   Sidebar: Full width (w-64)
   Carousel: 4 cards
   Grid: 4 columns
   Font: 14px body

🖥️ LARGE
   Width: 1280px+
   Carousel: 4 cards
   Grid: 5 columns
   Full spacing
```
